HOW TO SET UP AND RUN : 

1.Clone the repo
2.in directory backend : pip install -r requirements.txt

3. populate the db with : python init_db.py
4. start the backend server with : python main.py
6. the api will be live at http://localhost:3000 and the swagger at http://localhost:3000/docs

7. open the forntend index.html in a browser
