HOW TO SET UP AND RUN : 

1.Clone the repo
2.in directory backend :py -3.12 -m  pip install -r requirements.txt   (use python 3.12 for no problems with pydantic core ) 

3. populate the db with : py -3.12 init_db.py
4. start the backend server with : py -3.12 main.py
6. the api will be live at http://localhost:3000 and the swagger at http://localhost:3000/docs

7. open the forntend index.html in a browser
